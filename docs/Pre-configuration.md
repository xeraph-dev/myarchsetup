# Pre configuration



###### Install Rust

```shell
pacman -S rust
```



###### Install NVM

```shell
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```



After configuring zsh, type this:

```shell
nvm install node
```



###### Make usual directories

```shell
mkdir -p ~/.local/bin
mkdir ~/.wallpaper
mkdir ~/Download
mkdir ~/Documents
mkdir ~/Music
mkdir ~/Pictures
mkdir ~/Pictures/scrot
mkdir ~/work
```
